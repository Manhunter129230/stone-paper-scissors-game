# Stone Paper Scissors Game
A simple command-line program where you play Stone, Paper, Scissors against the computer. The computer's choice is generated randomly using the C built-in random functions. 


Features:

-> Play against computer opponent

-> Randomized computer moves

-> Simple text based interface


How to run:

-> Make sure that the C compiler(GCC or Clang) has been installed

-> Compile the program using the following command: gcc game.c -o game

-> Run the program using the following command: ./game

-> Choose your move when prompted: stone, paper, scissors (Case Insensitive)

(OR)

-> Download and run the game.exe file


How it works:

The program uses rand() function to generate the computer's move and compare it with the user's input using conditional logic to determine the winner
